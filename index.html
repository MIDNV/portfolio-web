Sobre mí

Estoy iniciando mi carrera como desarrollador web con foco en frontend, trabajando principalmente con HTML y CSS y construyendo proyectos prácticos como este portfolio para consolidar mis bases.

Me especializo en la creación de interfaces web estructuradas, accesibles y visualmente limpias, aplicando buenas prácticas de maquetación, organización de código y diseño responsive.

Actualmente estoy ampliando mis conocimientos hacia JavaScript para poder desarrollar experiencias web más dinámicas e interactivas, con el objetivo de incorporarme al entorno profesional del desarrollo frontend.

Proyectos

Este portfolio es mi primer proyecto completo, desarrollado desde cero utilizando HTML y CSS. Su objetivo es aplicar conceptos fundamentales como estructura semántica, diseño responsive y estilos modernos con CSS.

A través de este proyecto estoy practicando organización de código, diseño visual coherente y creación de interfaces web funcionales, con enfoque en buenas prácticas de desarrollo frontend.

Actualmente estoy ampliando este portfolio con nuevas secciones y mejoras progresivas, y planeo añadir proyectos adicionales que incluyan JavaScript para demostrar habilidades más avanzadas.
