# 🌐 Portfolio Web - MIDNV

Este es mi portfolio personal como desarrollador frontend en formación, creado desde cero utilizando HTML, CSS y JavaScript básico.

El objetivo de este proyecto es aplicar conocimientos de maquetación web, diseño responsive y buenas prácticas de desarrollo frontend, mientras construyo una presencia profesional en el mundo del desarrollo web.

---

## 🚀 Demo en vivo

🔗 https://oreipingo.github.io/portfolio-web/

---

## 🧠 Sobre el proyecto

Este portfolio ha sido diseñado y desarrollado como proyecto de aprendizaje continuo. Incluye mejoras progresivas en:

- Estructura HTML semántica
- Diseño moderno con CSS
- Efectos visuales (glassmorphism, hover states)
- Animaciones al hacer scroll
- Barra de progreso de lectura (scroll progress bar)
- Optimización visual tipo portfolio profesional

---

## 🎯 Objetivo

- Consolidar bases en HTML y CSS
- Practicar diseño de interfaces web modernas
- Construir un portfolio real y publicable
- Prepararme para aprendizaje de JavaScript y frameworks frontend

---

## 🛠️ Tecnologías utilizadas

- HTML5
- CSS3 (Flexbox, gradients, glassmorphism)
- JavaScript (Intersection Observer + scroll progress)
- GitHub Pages (deploy)

---

## ✨ Características

- Diseño responsive básico
- Animaciones suaves al hacer scroll
- Secciones estructuradas:
  - Sobre mí
  - Proyectos
  - Habilidades
  - Contacto
- UI moderna con estilo glassmorphism
- Botones de contacto directo (email + LinkedIn)
